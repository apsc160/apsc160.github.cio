DAQ Library and Simulator Installation

  • Installation
To install the DAQ library and simulator for Visual Studio on your local computer, we will download a zip file that contains all the required files, and then run an installation script that will adjust your Visual Studio properties.

Installation Instructions:

  1. Download the DAQ zip file from connect. This should create a zip file called DAQ_Student.zip in your downloads folder.
  2. Unzip the file by right-clicking on DAQ_Student.zip and selecting Extract all...
    Leave the checkbox Show extracted files when complete checked.


    Inside the unzipped folder, you should see a new subfolder named DAQ.



  3. Copy the subfolder DAQ to a permanent location. This could be somewhere on the Windows desktop, or directly in your C:\ drive. Once we finish installing the library, this folder should not be moved.



  4. Open the folder and double-click on install.bat


    If you get a security warning about opening a file with an unverified publisher, select Run.


    In Windows 10, if you get a SmartScreen warning message saying that it has prevented you from running the application, first click on More info and then Run anyway

    If you encounter an error that you cannot resolve, please either post a message on the discussion board with the output of the install script, or see a TA or instructor to help you complete the installation.
  6. Restart Visual Studio for the new settings to take effect.
