Lab 8: A Scrolling Message

  • Submission

Submission Instructions

  • Only submit one .c file containing your complete solution, including any bonus components.
  • Include sufficient comments for all important sections of your code.
  • Add a comment at top of your code indicating the following:
    • your name
    • your student number
    • your lab section (L1A, L1B, etc.)
    • the date
    • a brief description of what the program does, what is required as input, and what is generated as output
  • You may hand in your code anytime up to the deadline, which is common to all lab sections
  • Submit the source file using the usual submission procedure.
    In addition, note the following:
    • Absolutely no email submissions will be accepted. Do not leave your submission to the last minute.
    • Please verify that you are submitting your C files, i.e. you should not mistakenly upload any other files.

Submitting your Files

  • Create a ZIP file containing your source file for submission.
  • Go to https://www.ugrad.cs.ubc.ca/~apsc160/handin/.
  • Click on Start handin. Login with your undergrad account and password.
  • Select "Lab 8" from the list of assignments. Then click on "Choose File" or "Browse...", and select the ZIP file you created in the previous steps.
  • If you previously handed in a file for the current lab, check the "Override a previous submission" option, otherwise keep it unchecked. Then click on "Handin assignment". When presented with a confirmation dialog, read it and accept it.
  • Read the result of the handin submission. If it lists an error, post on the discussion forum, or ask a TA during office hours.
  • Once you have submitted the file, click on "Start handin" again, select "Lab 8" and click on "Check submissions". In the resulting page, check that the correct file is listed.
