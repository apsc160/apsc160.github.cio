Visual Studio

Microsoft Visual Studio is an integrated development environment (IDE) that allows you to edit, compile, link, run, and debug your C programs. It will also allow you to work with other programming languages such as C++, C#, and Visual Basic; but, we will not be exploring these languages in this course. The IDE is installed for your use on all machines in the CEME 2212 student lab.

If you wish to be able to develop programs using your own computer at home (and it's recommended that you do!), you will need to install Visual Studio 2017 Community Edition.

What's the difference between Visual Studio Enterprise and Visual Studio Community?

We will be using Visual Studio 2017 Enterprise Edition in the lab. This edition contains many more features than we really need in an introductory course. An alternate edition named Visual Studio 2017 Community is easier to install, requires less disk space, and can be downloaded for free from the Web via Microsoft.  The two editions look almost identical on the surface, and instructions provided for one edition will work equally well on the other.

Mac Users

How do I develop C programs on my Mac?

You have two options:

  • XCode - a native Mac application
  • Visual Studio running in a Windows 10 virtual machine

Of these two, the first is easier to set up but this will require you to become familiar with two different development environments: XCode and Visual Studio. Keep in mind that you will have to use Visual Studio in the lab each week. So, although the second option is more difficult to set up, it is the one we recommend.

How do I install XCode on my Mac?

If you wish to work with XCode, you can download it from the Mac App Store.

How do I install Visual Studio on my Mac?

As noted above, the recommended option is to install Visual Studio in a Windows 10 virtual machine on your Mac following these instructions. Note that you will need at least 8GB of memory and 32GB of free disk space.

Lastly, if you do not have a lot of disk space on your Mac (e.g. you bought a Mac Book Air with only 128GBs of Disk space), then you might find it a squeeze to install Windows, Visual Studio and all your OSX files onto that single disk. It is recommended in this case that you consider purchasing an external USB drive (1 and 2 TByte drives can be purchased for about $100 these days) and put some of your files onto that drive. This will of course mean that you have to take the drive with you.

Sophos Anti-Virus Software

Many of you will have rushed out over the summer and bought a shiny new laptop before starting UBC, perhaps a new Mac or a Windows-based machine. Even if you didn't and are using an older computer, it's important to keep your computer safe from viruses and other threats when going on-line. To help you with this, UBC has an agreement with Sophos Corporation, that allows all students registered at UBC to download Sophos anti-virus software for free.

It really is a very good package and works seamlessly with Windows (or a Mac with Windows installed on it - as some of you will do in this course) to protect your system from attack. If you have a reputable anti-virus software package installed on your computer and are happy with it, that's fine; continue to use it. However if you have a "less reputable" one, or worse, none at all, then you are strongly advised to install the Sophos software. You can download the software by visiting https://it.ubc.ca/software-downloads and clicking on the download button. You will need your UBC campus wide login (CWL) to start the download. Last year we had a student whose computer was so infected with viruses and malware pop-ups that her machine became virtually unusable. It took her a long time (and she lost some grades) to get it straightened out.

Security Practices

Good security is both a mindset and a process; it is not a product.  Simply having anti-virus protection on your laptop is an important step in this process.  Avoid clicking on links that you do not trust.  Avoid opening attachments in e-mails that come from people you don't know, or for which the context of the message is not what you might expect.  A big security problem these days is ransomware.  This occurs when you click on a link placed by a hacker, and the link intentionally encrypts files on your machine.  Once this happens, there are only two solutions: (a) restore your files from a recent backup (and you did backup the files on your laptop or home machine recently, right?), or (b) pay the ransom to the hacker who will (usually) give you the decryption key to un-encrypt your files.  Many organizations/users who didn't have a recent backup have had to do (b); otherwise, their files would be lost forever.  Do not share accounts and passwords.  Choose good passwords.  Remember to sign out when you're done.  Make use of disk encryption features built-in to your operating system; then, if your laptop is stolen, no one will be able to read the files on your machine.  Use common sense.  Use a password on your smartphone.

In case you're interested, UBC has some excellent security and privacy guidelines that are available at this URL: 
http://cio.ubc.ca/sites/cio.ubc.ca/files/documents/standards/IS%20Standards%201-21.pdf. 
Many of these policies are required by UBC staff and faculty members to protect people's personal information, to secure UBC files, and to use best practices in computing.
